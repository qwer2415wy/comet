​贊助後聯繫伺服主

最近出現有心人士惡意用本伺服器網址至他人頻道洗頻，在此澄清本伺服器不會未經允許在私人地盤進行任何宣傳行為。
​長期開發與架設伺服器需要耗費大量時間、金錢與軟硬體設備，如果願意提供贊助，本人LogoCat 不勝感激。​

自動化處理
​贊助管道：

透過此種方案贊助，不必聯絡管理員
綠界科技：將自動化處理你的贊助，​可以自由輸入贊助金額，
​第一時間「發送贊助廣播」、「獲得村民幣回饋」，方便又快速。
​繳費支援 超商繳款 / 信用卡 / 網路ATM / ATM櫃員機
綠界科技 超商繳費教學 ( 7-11、全家、萊爾富、OK )

人工處理
​贊助管道：

贊助完後需要聯繫伺服主，各平台在這時會稍有不同：
( 如果只是想知道「如何贊助」，請參考 這個頁面 )
使用8591商品下方的留言區即可填寫資料。
超商繳款之後別忘了按下「確認繳費」按鈕，伺服器才會收到通知。
​確認繳費按鈕在，8591網頁左下→我是買家→我的訂單→完成付款  (按鈕位置示意圖)

 * 8591的通知系統常常慢半拍，為了能更快幫您處理，贊助完後歡迎用下方式聯絡我：
​
1. Discord私訊  ( 請先加入廢土社群後，私訊  LogoCat#2523。未加入廢土社群會被系統自動擋下訊息 )
2. 傳送email ( admin@mcfallout.net ) 告知本人
3. 選擇傳送簡訊 或 致電本人手機  ( 8591下單後可取得號碼 )
LINE Pay：
以轉帳功能贊助後，使用以下方式聯絡本人，

1. Discord私訊  ( 請先加入廢土社群後，私訊  LogoCat#2523。未加入廢土社群會被系統自動擋下訊息 )
2. 傳送email ( admin@mcfallout.net ) 告知本人
(Line Pay會要求加入好友才能傳LINE訊息，可能會被系統自動擋下，請記得用以上方式聯絡)
街口支付：
不必額外聯絡本人，因贊助時填寫的「備註欄」有你的資料。入帳後本人會收到含有您ID的簡訊，會幫您處理。
歐付寶：
不必額外聯絡本人，因贊助時填寫的「備註欄」有你的資料。入帳後本人會收到系統email，會幫您處理。

2. 傳送email ( admin@mcfallout.net ) 告知本人
微信支付：
請使用以下方法，將您的匯款成功紀錄傳送給本人。
[1] Discord私訊  ( 請先加入廢土社群後，私訊  LogoCat#2523。未加入廢土社群會被系統自動擋下訊息 )

Paypal：
