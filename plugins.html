開源插件

使用時請勿做為贊助回饋或營利手段，以任何形式使用碼源時，您即同意此規章，
做為贊助回饋或營利手段時，​願意負擔違反智慧財產權連帶之法律責任。
若連結失效，代表該插件已從 Github.com 公開頁面移除，移除原因為部分插件屢次遭非法使用、作為贊助營利。
移除部分，僅做廢土開發紀錄與插件發想用途。如造成任何不便，本人實屬無奈，敬請見諒。
  • BetterWeather A plugin that broadcasts the weather around servers.
  • BetterPlayerLimit A plugin that makes the player limit based on claim size.
  • MoreTraderPlugin A plugin that makes Wandering Trader spawning based on the numbers of players.
  • ChatManager A plugin that makes  ChatControlPro channel blocking more comfortable.
  • BetterEntityDrop A plugin that issues commands or drops customized skin head when mobs get killed.
  • BetterWarpSign A plugin that makes signs more useful including warping and text telling.
  • RaidFixer A plugin that resets the raid fix by using reflection.
  • SAMLTweaks A plugin that makes iron golem farm more computationally affordable for servers.
  • MushroomMaker​ A plugin that transfers mycelium blocks into mushroom fields.
  • VillagerTradeFix A plugin that fixes villager's trade maximum time glitch through 1.13->1.14 update.
  • SilentMute​ A plugin that generates customizable fake messages and commands messages to statisfy the griefer.
  • AACTweaks​ A plugin that makes AAC more compatible with latest minecraft.
  • BetterSpawner A plugin that manages the spawner with balanced mechanism.
  • BetterBond A plugin that lets players to deposit money and exp point into items.
  • AutoBanMCLeaks A plugin that automatically bans mcleaks accounts with customizable commands.
  • WarpDescription A plugin that adds warp description while player using /warp
  • DiscordTweaks A plugin that revises the Discord-bot's responses.
  • LobbyWaiting A plugin that sends player to lobby before shutting down.
  • NoCraftingTablePlugin A plugin that disables crafting table to challenge players.
  • CoreProtectTweaks​ A plugin that tweaks on CoreProtect to stop the error spamming.
  • BlockLlama A plugin that blocks low hp llama/donkey/mule from portal teleporting due to vanilla duplication bug.
  • BetterMelon A plugin that tweaks on melon broken by pistons.
  • SafarinetTweaks A plugin that fixes SafariNet's NBT gam-crashing bugs
  • ClanTweaks A plugin that makes player chat in clans without using commands.
  • GesuitRandomWarp A plugin that randomly teleports player to a gesuit-warps.
  • BetterSponge A plugin that makes wet sponge clear lava and become dry sponge.
  • NBTChecker A plugin that checks multiple items' NBT data and perform multiple commands afterwards.
  • StreamCraft A broadcast system that helps streamers to interact with ingame players.
  • HideArmor A plugin that safely hides player's armor by editting packets. [Co-work with OowhitecatoO​]
  • Lunar A plugin that encourages the players to get engaged in specific works according to the lunar calendar.
  • EntityManager A plugin that controls the total amount of mobs and enhance the mob farm's performance.
  • FlyInClaim A plugin that gives player flying permission in their claim. Also enhances the experience inside the claim.
  • ServerTweaks Some tweaks that improves minecraft user experience.
  • Restorenature A plugin that periodically checks the chunks in worlds of its untouched time and respawn the resources.
  • CoinHiveMC A plugin that bridges Coinhive XMR miner with minecraft server.
  • Orereplacer A zero burden plugin that ends Xray-mining forever.
  • Bindwithvanish Enchant a binding curse item with vanish curse enchanted book could remove it, and vice versa.
  • Noafkfishing No more game breaking 24 hours afk fishing players.
  • Elytrasummon Let player summon Elytra by recreating the find experience.
  • CustomExplosion A plugin that could customize the explosion in minecraft such as radius, block breaking, block transformation.
  • DarksoulsMC Make minecraft looks like Darksouls. The process is done within the map generation.
  • Falloutcraft Fallout System in Minecraft (Radiation, Fatigue, Thirst, Hunger)
  • Newbiefactions A plugin that making factions with certain condition free from raiding.
  • CyberWorld Procedurally generating complicated cyberpunk city in minecraft.
    ​Automatically merging buildings into mega-structure. Generating the building with sub-structure such as signs, illegal building, etc.
  • DynMarket A plugin that sets the prices of items dynamically
  • Religion Adding religion and god-phrasing behaviors into minecraft.
  • Paytax Minecraft essentials /pay now will charge 7% tax.
Special Thanks to developers : 
  • SturdyTether A plugin that enables players to leash more types of living entities. [Independent work of OowhitecatoO​]
  • ArmorStandPlusLite A plugin that enables players to install the hand parts with sticks. [Independent work of OowhitecatoO​]
  • ​MessageAnyWhere Send chat message to anywhere you want. [Independent work of OowhitecatoO​]
